
На этапе разработки схемы базы данных необходимо выполнить анализ предметной области, т.е.~определить объекты предметной области и~связи между этими объектами. Опишем предметную область, которая будет использована в~нашем проекте.

Расписание занятий представляет собой таблицу, где указаны название дисциплины, преподаватель, ведущий занятие, время иместо проведения занятия, а~также студенты, которые должны посещать данное занятие.
\par\medskip Объекты предметной области:
\begin{MySpisok1}
\item Занятие (урок);
\item Учебная группа;
\item Преподаватель;
\item Дисциплина;
\item Кафедра;
\item Факультет;
\item Аудитория;
\item Корпус;
\item Время занятия;
\end{MySpisok1}
\par\medskip Определения и~свойства объектов:
\begin{MySpisok1}
\item  Занятие~--- урок в~учебном заведении. Свойства объекта <<занятие>>:
\begin{itemize}
\item Тип занятия: Возможные значения: лекция, практическое занятие, семинар, лабораторная работа и т.д.
\begin{description}
\item[Лекция] --- устное систематическое и последовательное изложение материала по какой-либо теме, разделу учебной дисциплины, а также письменная запись такого изложения.
\item[Практическое занятие] --- учебно-практическое занятие, на котором происходит решение и рассмотрение конкретных задач и примеров по какой-либо теме, разделу учебной дисциплины; проведение практических работ для контроля успеваемости учащихся.
\item[Семинар] --- учебно-практическое занятие, на котором происходит обсуждение сообщений, докладов и рефератов, выполненных учащимися по результатам учебных или научных исследований под руководством преподавателя.
\item[Лабораторная работа] --- учебно-практическое занятие, на котором студенты получают навыки экспериментальной работы, умение обращаться с приборами, самостоятельно делать выводы из полученных опытных данных.
\end{description}
\item Номера учебных групп студентов, обучающихся предмету.
\item ФИО преподавателя, ведущего занятие.
\item Номер аудитории, в которой происходит занятие.
\item Время (день, неделя и время начала и конца) занятия.
\end{itemize}
\item Учебная группа --- группа студентов, обучающихся в учебном заведении, Свойства объекта <<учебная группа>>:
\begin{itemize}
\item Номер курса.
\item Номер факультета.
\item Номер кафедры.
\item Номер группы на кафедре.
\item Размер группы.
\item Учебный план, по которому обучаются студенты группы.
\end{itemize}
\item Преподаватель --- человек, преподающий какую-либо дисциплину. Свойства объекта <<преподаватель>>:
\begin{itemize}
\item ФИО.
\item Учёное звание.
\item Кафедра, к которой приставлен преподаватель.
\item Контактные данные.
\item Дисциплины, которым может обучать преподаватель.
\end{itemize}
\item Дисциплина --- предмет, которому обучают студентов.
\begin{itemize}
\item Название дисциплины.
\item Количество преподаваемых часов по плану.
\item Тип дисциплины по плану (общие, либо специальные дисциплины)
\end{itemize}
\item Кафедра --- подразделение высшего учебного заведения. Свойства объекта <<кафедра>>:
\begin{itemize}
\item Ф.И.О заведующего кафедрой.
\item Факультет, к которому присоединена кафедра.
\item Контактные данные и другая информация.
\item Возможные направления подготовки.
\end{itemize}
\item Факультет~--- подразделение высшего учебного заведения. Свойства объекта <<факультет>>:
\begin{itemize}
\item Название.
\item Ф.И.О декана.
\item Контактные данные и другая информация.
\end{itemize}
\item Корпус~--- отдельное строение среди нескольких подобных или обособленная большая часть здания. Свойства объекта <<корпус>>:
\begin{itemize}
\item Название корпуса.
\item Адрес и контактная информация.
\end{itemize}
\item Время~--- параметр, определяющий даты и время проведения занятий. Свойства объекта <<время>>:
\begin{itemize}
\item Время начала и конца занятия.
\item День недели.
\end{itemize}
\end{MySpisok1}

\par\medskip Предполагается, что данную базу данных будут использовать три категории пользователей: диспетчер, преподаватель и студент.\\[6pt]
Диспетчер должен иметь возможность решать следующие задачи:
\begin{MySpisok2}
\item Ввод и редактирование данных о занятиях.
\item Просмотр данных о свободных аудиториях.
\item Определение количества студентов, планируемых посещать занятие, с целью совмещения их с вместимостью аудитории.
\item Подготовка к выводу на печать расписания занятий.
\end{MySpisok2}
Преподаватель и студент должны иметь возможность решать следующие задачи:
\begin{MySpisok2}
\item Просмотр расписания занятий в требуемой пользователем форме.
\item Подготовка к выводу на печать расписания занятий в требуемой пользователем форме.
\end{MySpisok2}
