{\bf \underline {Биллинговая система}} - программный комплекс, осуществляющий учет объема потребляемых абонентами услуг, расчет и списание денежных средств в соответствии с тарифами компании.

Клиент заключает договор с сотовым оператором. В договоре указываются абоненты, которые будут пользоваться услугами, предоставленными сотовым оператором. В большинстве случаев клиент, будет являться и единственным абонентом по данному договору. Однако возможен случай корпоративной связи. Каждому абоненту выдается лицевой счет, к которому привязан уникальный телефонный номер. При внесении абонентом денежных средств, они будут зачислены на лицевой счет абонента. Средства могу перемещаться лицевыми счетами абонентов. В договоре так же указывается выбранный тарифный план. При использовании абонентом услуг, предоставленных сотовым оператором, средства будут списаны с лицевого счета. К услугам сотового оператора относятся звонки и отправка коротких текстовых сообщений (sms), MMS, Internet. Расчет расхода денежных средств производится на основании объема потребленных услуг и их стоимости. Стоимость указывается в выбранном тарифном плане. При необходимости абонент может изменить тарифный план на любой активный тарифный план, предоставляемый оператором сотовой связи.

{\bf \underline {Абонент}} - лицо, использующее услуги сотового оператора.
\begin{itemize}
	\item	ФИО
	\item	Дата рождения
	\item	Телефон для связи
	\item Банковские реквизиты
	\item	Место работы
	\item	Место проживания
	\item Паспорт
\end{itemize}

{\bf \underline {Лицевой счет}} - счет, привязанный к телефонному номеру абонента. На него поступают пополнения средств. С него же списываются средства, израсходованные абонентом.
\begin{itemize}
	\item	Состояние
	\item	Размер кредита
	\item	Баланс
\end{itemize}

{\bf \underline {Тарифный план}} - это набор условий, в соответствии с которыми провайдер предоставляет клиентам свои товары и услуги. В тарифном плане указывается перечень возможных услуг и порядок определения их стоимости.
\begin{itemize}
	\item Активность
	\item Название тарифного плана
	\item	Стоимость подключения
	\item	Абонентская плата
	\item	Тип абонентской платы
\end{itemize}

{\bf \underline {Тарифный план по договору}}-тарифный план, выбранный лицом, заключившим договор с сотовым оператором.

{\bf \underline {Операция}} - услуга, предоставляемая оператором сотовой связи абоненту.
\begin{itemize}
	\item	Название
	\item	Единица тарификации
\end{itemize}

{\bf \underline {Договор}}
\begin{itemize}
	\item	Номер договора
	\item	Дата подписания
	\item	Статус договора
\end{itemize}

{\bf \underline{Абоненты по договору}}- абоненты, указанные в договоре.
\begin{itemize}
	\item	Телефонный номер
\end{itemize}

{\bf \underline {База клиентских операций}} - история всех совершенных абонентом операций с момента заключения договора.
\begin{itemize}
	\item Дата последней совершенной операции
	\item Все операции совершенные пользователем
\end{itemize}

{\bf \underline {Пополнение}} - запись о поступление средств.
\begin{itemize}
	\item	Сумма
	\item	Источник
	\item	Номер чека
\end{itemize}

{\bf \underline {Операция в тарифном плане}} - операции, имеющиеся в тарифном плане.
\begin{itemize}
	\item	Стоимость
\end{itemize}

{\bf \underline {Расход}} - запись о списание средств.
\begin{itemize}
	\item	Сумма
\end{itemize}
